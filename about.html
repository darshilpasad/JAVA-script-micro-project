About Ocean Pollution Monitoring Dashboard

Project Overview

The Ocean Pollution Monitoring Dashboard is a web-based application that provides real-time marine information using the Open-Meteo Marine API. It displays sea temperature, wave height, wind conditions and wave direction for selected coastal cities.

Objectives

  • Monitor ocean conditions in real time.
  • Create awareness about ocean pollution.
  • Visualize marine data using charts.
  • Promote environmental sustainability.

Technologies Used

HTML5
CSS3
Bootstrap 5
JavaScript ES6
Chart.js
Fetch API (Async/Await)

API Information

This project uses the Open-Meteo Marine Weather API to fetch live marine weather data for coastal cities.

Sustainable Development Goals

SDG 13 - Climate Action

Monitoring marine conditions helps understand climate change and supports climate-related research.

SDG 14 - Life Below Water

Ocean monitoring contributes to protecting marine ecosystems and reducing pollution.

Benefits

  • Real-time marine monitoring.
  • Supports environmental awareness.
  • Helps researchers and students.
  • Easy-to-use responsive dashboard.
